6.PURPOSE OF A TOKEN:

OPL token which we offer during TGE is a currency.

There’s going to be two type of tokens, first one is a utility token, is the one that pre-sale participants can purchase on February 1, this token will be circulating on different exchanges, it is domestic currency for OnPlace echo system.

The second type of token will be a security token, this token will be backed by shares of private companies. The only way to purchase a security token is through OPL currency token.
OnPlace will be the legal private asset holder, and security token holders will be members of participation.
